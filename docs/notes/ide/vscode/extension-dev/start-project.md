# 开始一个`vscode插件开发`项目

> 相关网站:
>
> [Extension API | Visual Studio Code Extension API](https://code.visualstudio.com/api)

## 使用`yo`脚手架来构建项目

::: code-group

```shell [临时安装yo]
npx --package yo --package generator-code -- yo code
```

```shell [全局安装yo]
npm install --global yo generator-code

yo code
```

:::

> 上面全局安装了两个包
>
> - `yo`
> - `generator-code`  vscode扩展生成器



**构建项目**

```shell
yo code
```
